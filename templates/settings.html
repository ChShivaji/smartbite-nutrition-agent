How to Get Keys

Step 1: Sign up / Log in

Log in to your IBM Cloud platform at cloud.ibm.com.

Step 2: Create Watsonx Service

Search for "watsonx.ai" in the catalog. Choose the Dallas (us-south) region, select the Lite Plan (Free), and click Create.

Step 3: Create Watsonx Project

Launch Watsonx.ai Studio, select "New Project", give it a name, associate your machine learning service instance, and click Create.

Step 4: Copy Project ID

Inside your project, navigate to the Manage tab > General section, and copy the Project ID.

Step 5: Create Cloud API Key

In the IBM Cloud console top menu, go to Manage > Access (IAM) > API Keys. Click "Create an IBM Cloud API key", name it, copy the secret key, and paste it here.
